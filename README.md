# Sistema Integral de Gestión Personal

## Taller de Programación

Este proyecto es un sistema académico desarrollado en la asignatura Taller de Programación para demostrar la implementación de un sistema de gestión personal aplicando los conceptos vistos en clase.

El sistema permite registrar ingresos, gastos, tareas y metas personales, facilitando el control de la información y el seguimiento de objetivos.


## Objetivo

Desarrollar un sistema de consola que permita administrar finanzas personales, tareas y metas, aplicando principios de Programación Orientada a Objetos, estructuras de datos y organización modular del código, utilizando GitHub para el control de versiones.


## Alcance

El sistema permitirá:

- Registrar usuarios
- Registrar ingresos
- Registrar gastos
- Calcular el balance disponible
- Registrar tareas
- Marcar tareas como completadas
- Crear metas personales
- Mostrar el porcentaje de avance de cada meta

El sistema se implementará en consola y almacenará los datos temporalmente en memoria.


## Requerimientos funcionales

El sistema debe permitir:

- Registrar usuarios
- Registrar ingresos
- Registrar gastos
- Calcular el balance financiero
- Registrar tareas
- Marcar tareas como completadas
- Crear metas personales
- Mostrar el progreso de las metas
  

## Requerimientos no funcionales

El sistema debe:

- Ejecutarse en consola
- Manejar errores de entrada del usuario
- Estar organizado en módulos
- Implementar principios de Programación Orientada a Objetos
- Mantenerse en un repositorio GitHub
  

## Estructura del proyecto

### Descripción de los módulos

**main.java**  
Archivo principal del sistema. Contiene el menú del programa y controla el flujo general de la aplicación.

**usuario.java**  
Contiene la clase Usuario y la gestión de la información del usuario.

**finanzas.java**  
Contiene las clases Ingreso y Gasto, así como las funciones relacionadas con el registro financiero.

**tareas.java**  
Contiene la clase Tarea y las funciones para registrar y gestionar actividades.

**metas.java**  
Contiene la clase Meta y las funciones relacionadas con la creación y seguimiento de metas.


## Tecnologías utilizadas

- Java
- Git
- GitHub
  

## Ejecución del sistema
ara ejecutar el sistema: main. java. Este archivo controla el flujo del sistema.
